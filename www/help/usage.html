Using the NZMapConv application

Converting coordinates

To convert coordinates simply enter the coordinate into the appropriate box (according to the type of coordinate), and then press the Enter key. The corresponding values in other coordinate systems will be shown in each box. If the coordinates cannot be computed (if the position is outside the valid range for them) then the box will be left blank.

You can only do a conversion when the entered coordinate is valid. So for example when you are typing in an NZMG coordinate you cannot do a conversion until both the easting and northing are entered. NZMapConv indicates that the entered coordinate is incomplete or invalid by colouring the background of the box.

The application is tolerant of different ways of entering coordinates. For example NZMS260 map references can be entered with or without spaces between the various components (eg "T23537191", "T23 537191", and " T23 537 191 " are all valid. Projection coordinates can be entered with the easting or northing first. Similarly latitudes and longitudes can be in either order, and can include just decimal degrees, degrees and minutes, or degrees, minutes, and seconds.

The format of the converted coordinates can be altered by changing the configuration settings

Reviewing previous coordinates

You can review previously converted coordinates using the "Previous" and "Next" buttons.

Changing configuration options

The application provides options for changing the format of the converted coordinates and for changing the layout of the coordinate conversion display. These can be accessed by clicking the "Options..." button. If this is not visible (there is an option to hide the buttons), then you can access it by clicking on the icon at the top left of the window (or pressing "Alt+space" and selecting "Options..." from the system menu).
